我想要开发一个在线知识库，方便我随时创建/创建/编辑 md文件。
它应该具有如下特点：
# 1、使用nuxtjs+tailwind css+shadcn-ui+PostgreSQL。
# 2、所见即所得的编辑markdown（Vditor插件），且应支持todolist（图片，如果可以的话）。
# 3、知识库由文件夹和md文件构成，在前端树状显示，后端存储你来推荐。
# 4、首次打开网页速度应该足够快。
# 5、系统能够快速从所有文档中检索出查找的内容，并以列表显示。
# 6、样式可以参考notion、outline等知识库系统，交互一定要流畅、系统一定要美观，用户体验要好，比如用户网速慢时先展示骨架。
# 7、只有管理员一个人可以登陆，并使用和访问系统内容。防止文件内容泄露。此部分只需要保持安全性即可，不考虑多用户和权限分配等。
